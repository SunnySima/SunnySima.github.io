常用命令

dashboard

显示当前系统实时数据（线程、GC、Runtime等信息）

用法

thread

查看线程信息，线程堆栈

用法

没有参数时显示所有线程信息

jvm

显示JVM信息

用法

sc

查看JVM加载的类信息

用法

sm

查看已加载类的方法信息

用法

jad

反编译指定已加载类

用法

monitor

方法执行监控，如：成功、失败次数，平均rt，失败率等

用法

watch

监控指定方法执行入参/出参、返回值、异常等信息

用法

trace

方法调用链路及各个链路耗时

用法

tt

方法执行数据的时空隧道，记录下指定方法每次调用的入参和返回信息，并能对这些不同的时间下调用进行观测

用法

quit

退出当前Arthas客户端，其他Arthas客户端不受影响

shutdown

关闭Arthas服务端，所有Arthas客户端全部退出

stop

和shutdown命令一致（推荐使用，避免误操作）
